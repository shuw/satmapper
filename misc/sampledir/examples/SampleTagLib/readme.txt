Sample custom tag library. 

This example contains a sample custom tag with an attribute and a variable. 
It illustrates the use of the Custom Tag Library development features in 
Forte for Java.

To view the details of an item in this directory, double click on the icon.
This will open a Customizer view (also accessible from the context menu).

Select "Generate Tag Handlers" from the context menu on the SampleTagLib object
to generate the Java classes where the tag's functional code will be placed.

For more information on creating tag libraries within Forte for Java, see
the following:

    The Building Web Applications Programming Guide can be found on the 
    Forte for Java Developer Resources website: 
    	http://www.sun.com/forte/ffj/resources/documentation/
